Open Source

Open Source Documentary Project Research / Documentation I'm trying to build a platform for Open Source Documentary Filmmaking. The idea of the model is that the original maintainers can set tasks, anyone can suggest contributions. If the contributions are accepted, you gain a voting stake and ownership stake over the project. This will only be suitable for certain types of film, but I think could be a valuable resource.
